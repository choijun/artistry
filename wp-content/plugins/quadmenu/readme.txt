== Description ==
[Demo](http://quadmenu.com/) | [Premium](https://codecanyon.net/item/quadmenu-responsive-wordpress-mega-menu/21315450?ref=QuadLayers) | [Documentation](http://quadmenu.com/documentation/) | [Developers](http://quadmenu.com/documentation/developers/)

QuadMenu will allows you easily integrate the menu in your WordPress theme project. This megamenu automatically convert your existing menu or menus into a mega menu. 

This WordPress Mega Menu include many powerful features that will blow your mind. The customizable admin panel allow you to change the colors and menu layout easily at fingertips. This panel allows you to create an unlimited number of skins as it includes an instance for each theme location in your theme.

Live Customize

https://www.youtube.com/watch?v=LwywYyS3D2M

###Features:

> * Automatic and manual menu integration
> * Builded extending the standard WordPress Menu system
> * Supports multiple menu locations
> * Drag and Drop admin builder
> * Display WordPress Widgets in your menu
> * Support for child themes
> * Unlimited Menu Themes
> * Vertical Mega Menu
> * Horizontal Mega Menu
> * Offcanvas Mega Menu
> * Sticky Menu
> * Click or use the hoverIntent plugin to open links
> * Align links options
> * Float dropdown menus options
> * Customizable breakpoint for menu collapse
> * Show or hide items depending on the screen size
> * Support for Font Awesome icons
> * Google Fonts
> * Dropdown animations
> * Support for Visual Composer
> * Mega Menu
> * Icons Menu
> * Search Menu
> * Cart Menu

####Pro Features:

> * Tabs Menu
> * Login Menu
> * Register Menu
> * Social Menu
> * Carousel Menu

== Installation ==

1. Go to the Plugins Menu in WordPress
1. Search for "QuadMenu"
1. Click "Install"

[http://www.quadmenu.com/documentation/installation/](http://www.quadmenu.com/documentation/installation/)
