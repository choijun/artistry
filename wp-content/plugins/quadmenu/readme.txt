The first responsive mega menu designed by developers for developers. 

== Description ==

QuadMenu will allows you easily integrate the menu in your wordpress theme project. This menu automatically convert your existing menu or menus into a mega menu. This WordPress Mega Menu include many powerful features that will blow your mind. The customizable admin panel allow you to change the colors and menu layout easily at fingertips. This panel allows you to create an unlimited number of skins as it includes an instance for each theme location in your theme.

Documentation & Demo: [http://www.quadmenu.com/documentation](http://www.quadmenu.com/documentation)

###Features:

Automatic and manual integration
* Builded extending the standard WordPress menu system
* Supports multiple menu locations
* Drag and Drop builder
* Display WordPress Widgets in your menu
* Support for child themes
* Unlimited Menu Themes
* Horizontal and vertical layouts
* Sticky menu on scroll
* Click or use the hoverIntent plugin to open links
* Align links options
* Float dropdown menus options
* Customizable breakpoint for menu collapse
* Show or hide items depending on the screen size
* Mega Menu
* Icons
* Social
* Search
* Cart
* Login
* Google Fonts
* Support for Font Awesome icons
* Dropdown animations
* Support for Visual Composer

####Pro Features:

> * Tabs menu
> * Carousel menu
> * Login and register form
> * Social networks icons
