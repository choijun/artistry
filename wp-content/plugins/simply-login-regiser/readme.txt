Easy way to create login and register page by shortcode. It\'s very simply to use this  plugin.

== Description ==
Simply Login Regiser is a easy way to create login and register page by usring short code. for create login page use <code>[slr_login]</code> shortcode and for register page add <code>[slr_register]</code> code on page.

== Installation ==
1. Upload category-post-slider.zip to the /wp-content/plugins/ directory
2. Unzip simply-login-register.zip
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. You will find \'simply login register settings\' under setting menu in your WordPress admin panel.
