== Description ==

Integrates QuadMenu with the Divi theme. Requires QuadMenu and Divi.

This plugin will add all the settings to match the style and the layout of Divi theme.

Documentation & Demo: [http://quadmenu.com/divi/](http://quadmenu.com/divi/)

== Installation ==

1. Go to the Plugins page in WordPress
1. Search for "QuadMenu - Divi Mega Menu"
1. Click "Install"

After installation go to Appearance > Menus and create your mega menu.
